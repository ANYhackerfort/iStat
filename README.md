# iStat - Frontend

**iStat** is a simple yet powerful statistical tool designed for everyone—from beginners to seasoned professionals. Think of it like the **GarageBand** or **iMovie** of statistical analysis, enabling users to create production-ready analyses of any dataset with ease. Whether you're new to data analysis or a pro, iStat helps you explore data intuitively, without needing to write a single line of code.

### Current Features:
- **Drag-and-Drop Nodes**: Effortlessly build your analysis pipeline by dragging and dropping nodes.
- **Connect Nodes**: Seamlessly connect nodes to create data workflows that analyze, visualize, and process information.
- **Custom Axioms**: Ensures consistency and accuracy throughout the analysis, with built-in rules to maintain statistical integrity.
- **Optimized UI**: A highly intuitive, responsive interface built with TypeScript and React, designed for smooth performance across all screen sizes and touch devices.

More advanced features and enhancements are planned for future iterations to expand iStat’s capabilities. Join the iStat community today and transform how you approach data analysis!
